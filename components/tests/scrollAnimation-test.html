使用说明

如何使用

  1. 引入 CSS 文件: <link rel="stylesheet" href="../scrollAnimation/scrollAnimation.css">
  2. 引入 JS 文件: <script src="../scrollAnimation/scrollAnimation.js"></script>
  3. 初始化动画: initScrollAnimation();
  4. 对需要动画的元素添加相应的类

配置选项

initScrollAnimation('.my-selector', {
  threshold: 0.2,    // 元素可见20%时触发
  activeClass: 'visible',  // 自定义激活类名
  once: false        // 元素离开视口后重置动画
});
