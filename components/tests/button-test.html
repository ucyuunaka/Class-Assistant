高级按钮功能

这里展示了本次优化后的各种高级按钮功能。

计数器按钮 (Badge Button)

计数器按钮在右上角显示一个数字标记，常用于通知提醒或数量指示。

// HTML代码 <button class="btn btn-primary btn-badge" data-badge="5">未读消息</button> // JavaScript代码 // 动态设置/更新徽章数量 ButtonUI.setBadge('#my-button', 10);

开关切换按钮 (Toggle Button)

开关切换按钮可以在两种状态之间切换，非常适合开启/关闭功能的场景。

// HTML代码 <button class="btn btn-toggle" data-on-text="已开启" data-off-text="已关闭"></button> // JavaScript代码 // 设置切换按钮状态 ButtonUI.setToggleState('#my-toggle', true); // 设置为激活状态 // 监听状态变化 document.querySelector('#my-toggle').addEventListener('toggle', function(e) { console.log('当前状态:', e.detail.isActive ? '开启' : '关闭'); });

自适应文本颜色

根据按钮背景色自动调整文本颜色，确保在任何背景下文本都清晰可见。

// HTML代码 <button class="btn" data-auto-color="true" style="background-color: #3498db;">蓝色背景</button> // JavaScript代码 // 手动触发文本颜色更新 ButtonUI.updateTextColor(document.querySelector('#custom-btn'));
