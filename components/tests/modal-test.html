模态弹窗组件测试

基本模态弹窗

测试基本的模态弹窗功能，包括不同大小和样式。

// 显示基本模态弹窗 showModal("基本模态弹窗", "这是一个基本的模态弹窗示例内容。"); // 显示小型模态弹窗 showModal("小型模态弹窗", "这是一个小型的模态弹窗。", [], { size: "sm" }); // 显示大型模态弹窗 showModal("大型模态弹窗", "这是一个大型的模态弹窗，适合显示更多内容。", [], { size: "lg" }); // 显示全屏模态弹窗 showModal("全屏模态弹窗", "这是一个全屏的模态弹窗，适合展示重要内容。", [], { size: "fullscreen" });

确认模态弹窗

测试需要用户确认的模态弹窗，包括基本确认、警告确认和危险操作确认。

// 显示基本确认弹窗 showConfirmModal( "确认操作", "您确定要执行此操作吗？", function() { console.log("用户点击了确认"); } ); // 显示警告确认弹窗 showWarningModal( "警告确认", "此操作可能导致数据变更，确定继续吗？", function() { console.log("用户确认了警告操作"); } ); // 显示危险操作确认弹窗 showDangerModal( "危险操作确认", "此操作将永久删除数据，无法恢复！确定继续吗？", function() { console.log("用户确认了危险操作"); } );

状态反馈模态弹窗

测试用于显示操作结果的模态弹窗，如成功或错误提示。

// 显示成功弹窗 showSuccessModal( "操作成功", "您的数据已成功保存！", function() { console.log("用户关闭了成功弹窗"); } ); // 显示错误弹窗 showErrorModal( "操作失败", "保存数据时发生错误，请重试。", function() { console.log("用户关闭了错误弹窗"); } );

自定义按钮弹窗

测试带有自定义按钮的模态弹窗，可以根据需求设置按钮数量、文本和样式。

// 显示带有自定义按钮的弹窗 showModal( "自定义按钮弹窗", "这个弹窗有多个自定义按钮，每个按钮都有不同的样式和功能。", [ { text: "取消", type: "secondary", handler: function() { console.log("用户点击了取消"); } }, { text: "保存草稿", type: "primary", handler: function() { console.log("用户点击了保存草稿"); } }, { text: "发布", type: "success", handler: function() { console.log("用户点击了发布"); } } ] ); // 显示带有居中按钮的弹窗 showModal( "居中按钮弹窗", "这个弹窗的按钮居中对齐显示。", [ { text: "知道了", type: "primary", handler: function() { console.log("用户点击了知道了"); } } ], { footerAlign: "centered" } );

现有模态窗口管理

测试管理已存在于DOM中的静态模态窗口的新功能：showExisting 和 hideExisting。

// 显示已存在于DOM中的静态模态窗口 Modal.showExisting(document.getElementById('static-modal')); // 带选项显示已存在于DOM中的静态模态窗口 Modal.showExisting(document.getElementById('static-modal'), { closeOnBackdrop: true, onOpen: function() { console.log('静态模态窗口已打开'); }, onClose: function() { console.log('静态模态窗口已关闭'); } }); // 隐藏已存在于DOM中的静态模态窗口 Modal.hideExisting(document.getElementById('static-modal'));
