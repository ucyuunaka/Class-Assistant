通知组件测试

基本通知类型

测试不同类型的通知样式和行为。点击下面的按钮触发不同类型的通知提示。

// 显示成功通知 showNotification("操作已成功完成！", "success"); // 显示错误通知 showNotification("操作失败，请重试", "error"); // 显示警告通知 showNotification("请注意，此操作可能有风险", "warning"); // 显示信息通知 showNotification("这是一条普通信息提示", "info");

持续时间测试

测试不同持续时间的通知。默认持续时间为3秒，您可以自定义持续时间。

// 显示短时间通知 showNotification("这是一条短时间通知", "info", 1000); // 显示默认时间通知 showNotification("这是一条默认时间通知", "info"); // 显示长时间通知 showNotification("这是一条长时间通知", "info", 6000);

"功能开发中"通知

测试显示功能开发中的通知。您可以自定义功能名称。

// 默认开发中通知 showDevelopingNotification(); // 自定义功能开发中通知 showDevelopingNotification("课程导出");

多个通知测试

测试同时显示多个通知的堆叠和排列效果。

// 显示多个通知 showNotification("第一条通知", "info"); setTimeout(() => showNotification("第二条通知", "success"), 300); setTimeout(() => showNotification("第三条通知", "warning"), 600); setTimeout(() => showNotification("第四条通知", "error"), 900);
