顶栏组件测试

这个页面展示了顶栏组件的不同使用方式和样式选项。

首页风格顶栏

首页风格顶栏有更大的字体大小和额外的按钮区域，适合用作网站首页的吸引点。 

const homeHeader = new Header('home-header-container', {
  isHomePage: true,
  title: '让你的学习生活更高效有序',
  subtitle: '课堂助手帮你管理课程安排、记录学习进度、提醒考试日期，让你专注于学习本身。',
  buttons: [
    { text: '查看我的课表', url: '#', isPrimary: true },
    { text: '考试倒计时', url: '#', isPrimary: false }
  ]
});

普通页面顶栏

普通页面顶栏适用于内容页面，字体较小，没有按钮区域。适用于成绩管理、课评速记等内容页面。 

const pageHeader = new Header('page-header-container', {
  isHomePage: false,
  title: '成绩管理',
  subtitle: '记录、分析并追踪你的学习成果'
});
